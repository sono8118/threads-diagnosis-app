## 4. 技術スタック（MVP最適化版）

### フロントエンド
- React 18 + TypeScript 5
- MUI v6（UIライブラリ）
- Zustand（状態管理）
- React Router v6（ルーティング）
- Vite 5（ビルドツール）
- Recharts（レーダーチャート）

### 診断ロジック
- フロント側で完結（ルールベース）
- sessionStorage（一時保存）

### 外部サービス
- Vercel（ホスティング）
- Google Analytics 4（計測）
- Google Tag Manager（タグ管理）
- UTAGE（メール登録・特典配信）

### 削除した技術（MVPでは不要）
- ❌ FastAPI（バックエンド）
- ❌ PostgreSQL/Neon（データベース）
- ❌ Resend（メール配信）

## 5. 成功指標（KPI）

### 定量的指標
- 月間診断受診者: 30-50人
- 診断完了率: 80%以上
- LP遷移率（CTR）: 25%以上
- 売上向上率: 140%

### 定性的指標
- ユーザーが自分の現在地を理解し、納得して商品を購入する
- 「診断を受けてよかった」と感じるユーザー体験
- 運用が自動化され、時間的自由が得られる
